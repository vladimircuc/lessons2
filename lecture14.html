>>> import matplotlib.pyplot as plt

What this is

Charts from a DataFrame

Every chart in this homework comes from calling .plot() on a DataFrame or Series, with a kind for the chart type. It hands back a matplotlib Axes object, which you then label and title. That Axes is what each function returns.

the pattern for every question

ax = data.plot(kind="...", ...), then ax.set_title(...), ax.set_xlabel(...), ax.set_ylabel(...), then return ax. The homework grades the chart, so the title and axis labels have to match exactly.

The [0] in the hint
With by=, the return is not a single Axes, it is a small collection, so the hint says grab the first with [0]. Newer pandas returns a labeled object where [0] raises a KeyError, and you use .iloc[0] instead. Both pull out the same Axes. Lecture 14 cheat sheet

      

        
The pattern

        
ax = data.plot(kind="...", ...)   # or fig, ax = plt.subplots()
ax.set_title("...")
ax.set_xlabel("...")
ax.set_ylabel("...")
return ax                          # every question returns the Axes

        
Reading and dates

        
pd.read_csv("f.csv", index_col="show_id")   set the index column
pd.to_datetime(df["date_col"])              text -> real dates
pd.to_datetime(df["d"]).dt.year             the year as a number
(pd.to_datetime(end) - pd.to_datetime(start)).dt.days   duration in days

        
Chart types

        
df.plot(kind="box")                      one box per numeric column
df.plot(kind="box", column="c", by="g")  one box per group (index [0] / .iloc[0])
df.plot(kind="scatter", x="a", y="b")    needs x and y column names
series.plot(kind="pie"); ax.set_ylabel("")   pie, clear the y label
series.plot(kind="bar", color="purple")  bar chart
ax.plot(x, y, color="blue", linestyle="--", label="...")   a line (matplotlib)

        
Grouping, bucketing, styling

        
df.groupby("year")["rate"].mean().reset_index()   avg per group, year as column
pd.cut(days, bins=[...], labels=[...], right=False)  sort numbers into ranges
series.value_counts().sort_index()        count per bucket, in order
ax.legend()   ax.grid(True)   ax.grid(axis="y")
ax.set_title("...", fontsize=14)          font size on title / labels
ax.text(i, value, round(value, 2), ha="center", va="bottom")   number over a bar

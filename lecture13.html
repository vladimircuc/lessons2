Tools for Q4

groupby, and counting distinct values

groupby splits the rows into groups by a column's value, then applies an aggregate to each group. You saw the shape rule in Lecture 12: one column name gives a Series, a list of names gives a DataFrame.

groupby_recap.py

      

      
The new aggregate for Q4 is nunique, which counts how many distinct values are in each group. Repeats only count once.

      
nunique vs count
count counts how many rows. nunique counts how many different values. If team A visited LA twice, count sees two but nunique sees one.

      

        
nunique.py

        
        

      

      
Also useful: agg
To get several aggregates at once, pass a list to .agg, like df.groupby("team")["points"].agg(["sum", "mean", "max"]). Not needed for Q4, but it is a core groupby move.

Tools for Q5

      
Building new columns

      
You add a column just by assigning to it: df["new"] = .... What goes on the right can be text built from other columns, or a True/False result from a comparison.

      

        
new_columns.py

        
        

      

      
text columns add like strings
df["a"] + "-" + df["b"] joins each pair row by row. The literal "-" is placed between them. A comparison like df["x"] > 1000 gives a whole column of True and False.

Tools for Q6

      
Sorting, dropping duplicates, taking the top

      
Three moves that go together. sort_values orders rows by a column, ascending=False for biggest first. drop_duplicates(subset=[...]) removes rows that match on the listed columns. head(n) keeps the first n rows.

      

        
sort_dedup_head.py

        
        

      

      
subset matters
drop_duplicates() with no subset only drops rows that are identical across every column. With subset=["price", "day"] it drops rows that merely match on those two, which is what Q6 wants.

Your source of truth

      
Lecture 13 cheat sheet

      

        
Grouping

        
df.groupby("x")["y"].sum()          total of y per group (Series)
df.groupby("x")["y"].mean()         average of y per group
df.groupby("x")["y"].nunique()      DISTINCT values of y per group
df.groupby("x")[["y"]].mean()       double brackets -> a DataFrame
df.groupby("x")["y"].agg(["sum","mean","max"])   several at once

        
Building new columns

        
df["new"] = ...                     assign to make a column
df["a"] + "-" + df["b"]             glue text columns row by row
df["stops"] != 1                    a True/False column
df["duration"] > 24 * 60            another boolean column

        
Sorting, dedup, top rows

        
df.sort_values("price")             ascending by default
df.sort_values("price", ascending=False)   biggest first
df.drop_duplicates(subset=["a","b"])  drop rows matching on a AND b
df.head(5)                          first 5 rows
df[["a","b","c"]]                   keep only these columns

        
Chaining (Q7 shape)

        
df[df["x"] == val]                  filter
   .sort_values("y", ascending=False)   sort
   .head(3)                          take top 3
   [["c1","c2","c3"]]               keep chosen columns
# each step feeds the next, left to right
